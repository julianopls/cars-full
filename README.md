# 🚗 Cars Full Stack

> Sistema completo de gerenciamento de carros desenvolvido com arquitetura full stack.

---

## ✨ Sobre o projeto

O **Cars Full Stack** é uma aplicação que permite realizar o controle completo de veículos, incluindo cadastro, edição, listagem e exclusão.

O projeto foi criado com foco em prática real de desenvolvimento, integrando **frontend + backend + banco de dados** em um único sistema funcional.
