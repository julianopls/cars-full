## ✨ Sobre o projeto

O **Cars Full Stack** é uma aplicação completa para gerenciamento de carros, permitindo realizar todas as operações essenciais de um sistema real:

- Cadastro de veículos  
- Visualização em lista  
- Atualização de dados  
- Remoção de registros  

Tudo isso com integração entre **frontend + backend + banco de dados**.
